Este proyecto consiste en una página web dedicada a la gastronomía, donde los usuarios pueden explorar diferentes platillos, 
recetas y culturas culinarias. La web busca ofrecer una experiencia visual atractiva y fácil de usar, mostrando información 
sobre comidas.
